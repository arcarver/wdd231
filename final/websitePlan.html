Site Name:"Dad Jokes"

I selected this name because it described what the web site was for. I figured it would also be searchable. Dadjokes.com would be a nice domain name.

Site Purpose:

The purpose of this site is to make people laugh. There is a lot of not great things happening right now in the world. I felt like cheering people up.

Senarios:

Questions this web site will answer:

  • What are some jokes about Bison?
  • Tell me a random joke.
  • Can I add my own jokes to the list of jokes?
  • What are some other rescources for jokes?
  • What are some resources that can help if jokes are not helping enough?
